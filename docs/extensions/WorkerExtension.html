Worker extension.

Worker extension created to make use of as3 workers easy.
Features:
  • Create background worker from main swf file, or loaded one.
  • Send and receive messages between workers.
  • Class aliases are automatically registered between workers. (to preserve parameter object type, instead of receiveng Object's.)
