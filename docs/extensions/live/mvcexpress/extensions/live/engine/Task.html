Classpublic class Task
InheritanceTask Object

Task represent one application logic block. It will be repeatedly and contingently executed by process.

Can be disabled/enabled. Can send messages. Then CONFIG::debug mode enabled - functions marked with Test metadata tag will be executed by process.
