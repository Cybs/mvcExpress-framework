getProxy()method
public function getProxy(injectClass:Class, name:String):Proxy

Get mapped proxy. This is needed to get proxy manually instead of inject it automatically. You might wont to get proxy manually then your proxy has dynamic name. Also you might want to get proxy manually if your proxy is needed only in rare cases or only for short time. (for instance - you need it only in onRegister() function.)

Parameters

injectClass:Class — Optional class to use for injection, if null proxyObject class is used. It is helpful if you want to map proxy interface or subclass.
 
name:String — Optional name if you need more then one proxy instance of same class.

Returns
